# Test

## Functions

### `assert`

Assert that two values are equal.

| Parameters | Type | Description         |
| ---------- | ---- | ------------------- |
| condition  | bool | Condition to assert |

### `equal`

Assert that two values are equal.

| Parameters | Type | Description |
| ---------- | ---- | ----------- |
| left       | any  | Left value  |
| right      | any  | Right value |

### `fail`

Eval an expression and check the error message pattern is matched.

| Parameters        | Type | Description            |
| ----------------- | ---- | ---------------------- |
| mixed_list        | list | Expression to evaluate |
| error_msg_pattern | str  | Error message pattern  |
