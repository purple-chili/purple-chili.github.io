# Matrix Functions

## `inv`(Pending)

Inverts a matrix.

| Parameters | Type   | Description      |
| ---------- | ------ | ---------------- |
| matrix     | matrix | Matrix to invert |

### `reshape`

Reshapes a collection into a matrix.

| Parameters | Type       | Description           |
| ---------- | ---------- | --------------------- |
| shape      | list       | Shape of the matrix   |
| collection | collection | Collection to reshape |
