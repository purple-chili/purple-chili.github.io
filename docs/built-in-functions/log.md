# Log

## Log Levels

This can be configured by command line option `-l` or `--log-level`.

## Functions

### `.log.debug`

Logs a debug message.

| Parameters | Type | Description    |
| ---------- | ---- | -------------- |
| message    | any  | Message to log |

### `.log.error`

Logs an error message.

| Parameters | Type | Description    |
| ---------- | ---- | -------------- |
| message    | any  | Message to log |

### `.log.info`

Logs an info message.

| Parameters | Type | Description    |
| ---------- | ---- | -------------- |
| message    | any  | Message to log |

### `.log.warn`

Logs a warning message.

| Parameters | Type | Description    |
| ---------- | ---- | -------------- |
| message    | any  | Message to log |
